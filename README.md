# ExpressJS 프로젝트 시작하기
1. 폴더 생성
2. `npm init -y`
3. `npm install express`
4. create `server.js`
### nodemon 설치
`npm install -D nodemon`
```json
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "start": "node server.js",
    "dev":"nodemon server.js"
  },
```
